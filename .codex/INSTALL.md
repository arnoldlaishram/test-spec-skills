# Test Spec Generator for Codex

To use the Test Spec Generator with Codex, you need to manually install the skills.

## Manual Installation

Tell Codex:

```
Clone https://github.com/final-run/test-spec-skills to ~/.config/codex/plugins/test-spec-gen, then add ~/.config/codex/plugins/test-spec-gen/skills to your skills path.
```

Alternatively, you can symlink the skills directory if you have local access.

## Usage

Once installed, you can use the `generate-test-spec` skill directly in your Codex environment.
